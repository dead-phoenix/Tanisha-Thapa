F1 Global Fandom Growth

Interactive data visualisation — D3.js, Google Trends, FastF1 API

Stitched together Google Trends, the FastF1 API, and 154+ manually compiled country-year event records into one comparable time series — solving Trends' window-normalisation problem with an overlapping-window scaling technique so separately-pulled windows line up instead of jumping at the seams.

Used OLS regression, paired t-tests, and correlation analysis to separate event-driven growth from on-track performance, isolating Drive to Survive's effect on fan interest across 14 countries.

Took the exploratory findings from Tableau and rebuilt them as a narrative-driven D3.js visualisation, designed through the Five Design Sheet method for a public, non-technical audience.

2025

Behavioural Stress Index (BSI)

Economic forecasting — PCA, Ridge Regression, walk-forward validation

Built an interpretable stress index from Australian Google Trends and ABS macroeconomic data, using PCA and Ridge Regression to identify which search behaviours — Centrelink, mortgage stress, food banks — actually track unemployment.

Wrapped the index in a walk-forward classifier that flags forward unemployment spikes ahead of the official statistics, evaluated strictly out-of-sample to avoid look-ahead bias.
